Faircode Blog Application Backend
I have created the backend section for blog application using Python Django framework in Pycharm.

Database management:MySQL to make models for Blog User and posting

Installed modules and packages:
* Django REST Framework: To handle operations for User User registration and for blog post to handle CRUD(create read update delete) opertaions
* Python-dotenv: reads key-value pairs from a .env file and can set them as environment variables
* Pymysql:To connect to a MySQL database using Python.
* djangorestframework-simplejwt: Simple JWT a JSON Web Token authentication plugin for the Django REST Framework.
* django-ckeditor: rich text editor which enables writing content directly inside of web page


Authentication:IsAuthenticated function from django rest framework permissions for User creation, login, logout, post creation, view, updation, deletion

Dockerization:
* Dockerfile: containing instructions for building your source code.
* docker-compose.yml
* requirements.txt: run pip freeze > requirements. txt when all packages are installed
